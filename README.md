This project focuses on analyzing the UCI Heart Disease dataset using data preprocessing and exploratory data analysis (EDA) techniques, followed by applying multiple machine learning models for classification.

The workflow includes handling missing values, feature scaling using Standardization, and visual analysis through various plots. Several classification algorithms such as Logistic Regression, K-Nearest Neighbors (KNN), Naive Bayes, Decision Tree, Support Vector Machine (SVM), and Random Forest are implemented and compared based on their performance.

The project highlights the importance of data preprocessing and demonstrates how different models perform on the same dataset.
